# Login Page Sederhana

Proyek ini adalah halaman login sederhana yang dibuat menggunakan **HTML**, **CSS**, dan **JavaScript (Vanilla JS)** tanpa menggunakan framework atau database.

Project ini dibuat sebagai bagian dari pembelajaran dasar pengembangan web, khususnya dalam memahami struktur halaman, styling, dan interaksi dasar menggunakan JavaScript.

---

## Fitur

- Form login sederhana (username & password)
- Validasi input dasar menggunakan JavaScript
- Tampilan responsif *(opsional, jika kamu buat)*
- Tanpa backend / database (front-end only)

---

## Teknologi yang Digunakan

- HTML5
- CSS3
- JavaScript (Vanilla JS)

---

```
project-folder/
```

## Catatan

- Project ini **tidak menggunakan database**
- Tidak ada sistem autentikasi yang sebenarnya
